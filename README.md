## 🚀 **How to Use**

1. Clone the repository:

```bash
git clone https://github.com/username/repository_name.git
````

2. Navigate to the program you want to run.

3. Compile using GCC or G++:

```bash
gcc program.c -o program
./program
```

Or for C++:

```bash
g++ program.cpp -o program
./program
```
